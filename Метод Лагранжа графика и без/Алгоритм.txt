Метода Лагранжа:

1. Дано n+1 точек (x0, y0), (x1, y1), ..., (xn, yn).
Это означает, что у нас есть набор из n+1 точек на координатной плоскости, где для каждой точки известны ее x-координата и y-координата.

2. Вычислить базисные многочлены Лагранжа:
Базисные многочлены Лагранжа - это специальные математические выражения, которые используются для построения интерполяционного многочлена.
Они определяются следующим образом:

    l0(x) = (x - x1)(x - x2)...(x - xn) / (x0 - x1)(x0 - x2)...(x0 - xn)
    l1(x) = (x - x0)(x - x2)...(x - xn) / (x1 - x0)(x1 - x2)...(x1 - xn)
    ...
    ln(x) = (x - x0)(x - x1)...(x - xn-1) / (xn - x0)(xn - x1)...(xn - xn-1)
Каждый базисный многочлен li(x) зависит от всех x-координат точек, кроме xi.
3. Вычислить интерполяционный многочлен Лагранжа:
Интерполяционный многочлен Лагранжа - это формула, которая позволяет вычислить значение функции в любой точке x,
используя известные значения y в опорных точках (x0, y0), (x1, y1), ..., (xn, yn):
    L(x) = y0l0(x) + y1l1(x) + ... + yn*ln(x)
Здесь yi - это y-координаты опорных точек, а li(x) - это базисные многочлены Лагранжа.

Таким образом, алгоритм метода Лагранжа позволяет построить интерполяционный многочлен,
который проходит через заданные опорные точки. Этот многочлен можно затем использовать для вычисления значений функциив промежуточных точках.
