Метод Зейделя - это разновидность метода простых итераций, которая обеспечивает более быструю сходимость.

Алгоритм метода Зейделя:

1. Выбираем начальное приближение x0 = (x01, x02, ..., x0n).
2. Вычисляем новое приближение x_new по формуле:
    x_new = (x1, x2, ..., xn), где
    x1 = g1(x1, x02, ..., x0n)
    x2 = g2(x1, x2, x03, ..., x0n)
    ...
    xn = gn(x1, x2, ..., xn-1)
    где G(x) = (g1(x), g2(x), ..., gn(x)) - вектор-функция, полученная из исходной системы уравнений.
3. Проверяем условие сходимости: если ||x_new - x0|| < ε, то x_new является решением, иначе переходим к шагу 2, заменив x0 на x_new.

Основное отличие метода Зейделя от простых итераций заключается в том, что при вычислении нового приближения x_new мы используем уже обновленные значения x1, x2, ..., xi-1, а не значения из предыдущего приближения x0.

Это позволяет добиться более быстрой сходимости по сравнению с методом простых итераций, особенно когда неизвестные сильно связаны между собой в системе уравнений.
